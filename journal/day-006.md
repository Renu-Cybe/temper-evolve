## ✅ 实现内容

### 1. 新增模块：`temper/heartbeat/`

**文件结构：**
```
temper/heartbeat/
├── __init__.py        # 模块入口
└── evolver.py         # 核心进化器（445行）
```

**核心类：`TemperEvolver`**

```python
class TemperEvolver:
    """心跳进化器
    
    定时触发四自行为：
    - 每60秒：自感知（资源检查 + 告警）
    - 每5分钟：自适应（参数调优建议）
    - 每1小时：自编译（代码扫描）
    """
```

### 2. 配置参数

| 参数 | 默认值 | 说明 |
|------|--------|------|
| `self_check_interval` | 60s | 自检间隔 |
| `adapt_interval` | 300s | 自适应间隔 |
| `repair_check_interval` | 3600s | 修复检查间隔 |
| `auto_repair_enabled` | False | 自动修复（安全优先）|
| `debug` | False | 调试输出 |

### 3. 集成到主系统

**修改文件：** `temper/__main__.py`

**新增：**
- 导入 `TemperEvolver, EvolverConfig`
- 初始化时创建进化器实例
- 新增 `/evolver` 命令
- shutdown 时自动停止进化器

### 4. 新增命令

```
/evolver        # 查看状态
/evolver start  # 启动进化器
/evolver stop   # 停止进化器
/evolver debug  # 切换调试模式
```
