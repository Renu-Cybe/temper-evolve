## What

实现自我记录能力，让 Temper 能够保存快照、对比变更、自动生成进化日志。

### 新增模块

```
temper/tools/
└── self.py              # 自我进化工具模块
```

### 新增工具

| 工具 | 功能 | 示例 |
|------|------|------|
| `self.snapshot` | 创建代码快照 | `{"tool": "self.snapshot", "args": {"files": ["temper.py"], "tag": "修改前"}}` |
| `self.diff` | 对比代码变更 | `{"tool": "self.diff", "args": {}}` |
| `self.log` | 生成进化日志 | `{"tool": "self.log", "args": {"title": "...", "description": "..."}}` |
| `self.list_snapshots` | 列出所有快照 | `{"tool": "self.list_snapshots", "args": {}}` |

### 快照系统设计

```python
# 快照目录结构

├── snap_20260330_215618/

    }
  ]
}
```

## 修复

### Bug: TemperOk 类不添加 ok 字段

**问题：** 当 `ok(value)` 传入 dict 时，没有添加 `ok: True` 字段。

```python
# 修复前
class TemperOk(dict):
    def __init__(self, value):
        if isinstance(value, dict):
            super().__init__(value)  # ❌ 直接复制，没有 ok 字段
        else:
            super().__init__({"value": value, "ok": True})

# 修复后
class TemperOk(dict):
    def __init__(self, value):
        if isinstance(value, dict):
            data = dict(value)
            data["ok"] = True  # ✅ 确保 ok 字段存在
            super().__init__(data)
        else:
            super().__init__({"value": value, "ok": True})
```

**影响：** 导致 `is_ok()` 判断失败，`self.diff()` 返回的结果无法正确解析。
