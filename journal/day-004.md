# Day 4 — 工具链组合与并行执行

**Time:** 2026-03-31
**Version:** v2.2
**Change:** 新增工具链组合和并行执行能力

---

## What

实现工具链组合，让 Temper 能够一次执行多个工具，支持串行和并行两种模式。

### 新增能力

| 模式 | 功能 | 示例 |
|------|------|------|
| `chain` | 串行工具链 | `[{"tool": "fs.read", ...}, {"tool": "fs.edit", ...}]` |
| `parallel` | 并行工具链 | `[{"tool": "fs.list", ...}, {"tool": "fs.read", ...}]` |

### 工具链执行流程

```python
# 串行模式（chain）
for tool in chain:
    result = execute(tool)
    if is_error(result):
        break  # 出错停止
    results.append(result)

# 并行模式（parallel）
with ThreadPoolExecutor() as executor:
    futures = [executor.submit(execute, tool) for tool in chain]
    results = [f.result() for f in futures]
```

### 输入格式

```json
{
  "chain": [
    {"tool": "self.snapshot", "args": {"files": ["temper.py"], "tag": "修改前"}},
    {"tool": "fs.edit", "args": {"path": "temper.py", "old_string": "...", "new_string": "..."}},
    {"tool": "self.diff", "args": {}},
    {"tool": "self.log", "args": {"title": "工具链测试", "description": "..."}}
  ]
}
```

或并行模式：

```json
{
  "parallel": [
    {"tool": "fs.list", "args": {"path": "temper/tools"}},
    {"tool": "fs.read", "args": {"path": "README.md"}},
    {"tool": "shell.run", "args": {"cmd": "git status"}}
  ]
}
```

### 问题

Temper 一次只能执行一个工具，复杂任务需要多轮对话：

```
👤: 创建快照
🤖: {"tool": "self.snapshot", ...}
👤: 修改代码
🤖: {"tool": "fs.edit", ...}
👤: 对比变更
🤖: {"tool": "self.diff", ...}
👤: 生成日志
🤖: {"tool": "self.log", ...}
```

### 解决方案

让 Temper 支持**工具链组合**：

```
👤: 执行完整进化流程
🤖: {"chain": [snapshot, edit, diff, log]}
✅: 一次完成4个步骤
```

```

  {"tool": "fs.read", "args": {"path": "CHANGELOG.md"}}
]}
✅ 结果: [{"content": "..."}, {"content": "..."}] (同时返回)
```

### 测试错误处理

```
👤 你: 执行会出错的链
🤖 Temper: {"chain": [

  {"tool": "fs.read", "args": {"path": "不存在.txt"}},  # 这里会失败
  {"tool": "fs.list", "args": {"path": "."}}  # 不会执行
]}
✅ 结果: [{"content": "..."}, {"error": {...}, "chain_stopped": true}]
```
